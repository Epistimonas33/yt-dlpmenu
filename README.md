YouTube/Media Downloader Script (Greek Interface)

Αυτό είναι ένα bash script με ελληνικό μενού που χρησιμοποιεί το yt-dlp και το ffmpeg για να κατεβάζει βίντεο, playlists ή ήχο από διάφορες πλατφόρμες.
Διαθέτει επιλογές για κατέβασμα σε υψηλή ποιότητα, συγκεκριμένη μορφή, ή μετατροπή σε MP3.
Το script ψάχνει αυτόματα για cookies του Firefox (ή διαβάζει από αρχείο) για πρόσβαση σε περιεχόμενο με περιορισμούς, και προσφέρει διαδραστικό μενού για εύκολη χρήση.

Λειτουργίες:

Κατέβασμα μεμονωμένου βίντεο (με επιλογή ποιότητας ή MP3)

Εμφάνιση διαθέσιμων μορφών βίντεο

Κατέβασμα ολόκληρης playlist

Κατέβασμα από αρχείο με λίστα URLs (.txt)

Αυτόματη χρήση cookies από Firefox (ή από αρχείο cookies.txt)

Απαιτήσεις: yt-dlp, ffmpeg
